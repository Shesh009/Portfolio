You can view the live version of the portfolio website on GitHub Pages by clicking the link below:

[Sheshu Akula's Portfolio](https://portfolio-bbe2.onrender.com/)

Alternatively, you can clone this repository and open the `index.html` file in your web browser to view it locally.

### Cloning the Repository

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
```

Then, open `index.html` in your preferred web browser.

## Project Structure

The repository contains the following files and directories:

```
portfolio/
│
├── docs/
│   └── Sheshu_Updated_Resume.pdf
├── images/
│   ├── myphoto.png
│   ├── linkedin.png
│   ├── github.png
│   └── ...
├── index.html
├── style.css

```

- `docs/`: Contains documents such as my resume.
- `images/`: Contains images used in the portfolio, including my profile photo and social media icons.
- `index.html`: The main HTML file for the portfolio website.
- `style.css`: The CSS file for styling the portfolio website.
- `README.md`: This file, containing information about the project.
